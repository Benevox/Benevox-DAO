# **App Name**: Benevox DAO Landing

## Core Features:

- Hero Section: Landing page section with headline, sub-headline, CTAs (call to action).
- Information Sections: Display of 'How It Works' and 'Why Benevox DAO?' sections with icons and clear explanations.
- Email Capture: Collect user email addresses to build waitlist for launch.

## Style Guidelines:

- Primary color: Dark Tech Blue (#1E3A8A) for a modern, trustworthy look.
- Background color: Very dark blue (#0F172A), near-black, for dark mode aesthetic.
- Accent color: Electric blue (#60A5FA), slightly desaturated, for highlights and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral feel.
- Simple line icons to illustrate steps and benefits.
- Responsive design using Flexbox/Grid for optimal viewing on all devices.
- Subtle scroll-reveal animations to draw attention as the user navigates the page.
